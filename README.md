# BLINKIT DASHBOARD

I created an interactive Blinkit Sales Dashboard using Power BI to analyze sales performance, item categories, outlet establishment, and customer ratings.

The dashboard includes KPIs such as:

* Total Sales
* Average Sales
* Item Count
* Average Rating

I used different visualizations like:

* Bar Charts
* Treemap
* Donut/Radial Charts
* KPI Cards

I performed data cleaning and transformation using Power Query and created relationships between datasets for accurate analysis. The dashboard helps in understanding sales trends, outlet performance, and item category insights in an easy and interactive way.

Tools & Skills Used:

* Power BI
* Power Query
* Data Cleaning
* Data Visualization
* Dashboard Design

* 
* DAX Basics
* Business Insights Analysis
